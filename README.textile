h1. Java & Android

This library is fully compatible with Android. The @pom.xml@ file has two profiles: 
* **target-java** is the default and use commons java dependencies(@org.json:json-20070829@ and @org.apache.httpcomponents:httpclient-4.0.2@)
* **target-android** is activable setting system property *keolis-rennes-api.version* to *android*. Then the dependencies will be switched to (@com.google.android:android-2.3.3@)

Please report bugs on "Github tracker":https://github.com/kops/keolis-java-api/issues/new or "contact us":http://www.google.com/recaptcha/mailhide/d?k=01Th60_7w3rxWuSJumsnqxfg==&c=REgbsYXndhO58POROxZGybu0F_Xu3JmR-wBRNbh8knE.

h1. Features

API version 2 only is supported. 

Fully implemented methods: 

* getbikestations
* getmetrostations
* getlinesalerts
* getrelayparks
