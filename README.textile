h2. Java & Android

This library is fully compatible with Android. The @pom.xml@ file has two profiles: 
* **java** is the default and use common java dependencies(@org.json:json-20070829@ and @org.apache.httpcomponents:httpclient-4.0.2@)
* **android** can be activated using command **mvn -Pandroid clean install**. Then the dependencies will be switched to android dependency (@com.google.android:android-2.1.2@)

Please ask for enhancements and report bugs on "Github tracker":https://github.com/kops/rennes-data-api/issues/new or "contact us":http://www.google.com/recaptcha/mailhide/d?k=01Th60_7w3rxWuSJumsnqxfg==&c=REgbsYXndhO58POROxZGybu0F_Xu3JmR-wBRNbh8knE.

h2. Features

API version 2 only is supported. 

Fully implemented methods: 

* getbikestations
* getmetrostations
* getlinesalerts
* getrelayparks
